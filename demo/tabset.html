Tabset without onHashChange event:

You can set this attribute in different ways:
in javascript: new Com.Tabset({'toggleOnHashChange' : false}); or just add "data-toggleOnHashChange" attribute in html: <ul class="tabset-content-ul" data-tabset="true" data-toggleOnHashChange="false"></ul>

Tabset without onHashChange event and with active tab:

You can set this attribute by just adding "data-active" attribute in html: <ul class="tabset-content-ul" data-tabset="true" data-toggleOnHashChange="false" data-active="second"></ul>

Tabset with events:

1. Disable tabset rendering by adding "data-renderOnInit" attribute in html
<ul class="tabset-content-ul" data-tabset="true" data-toggleOnHashChange="false" data-renderOnInit="false"></ul> 2. Set id attribute on tabset in html
3. Now you can add events in javascript
3.1 Use Com.GetTabset('id'); function for getting tabset by id.
Com.GetTabset('id'); 3.2 Add Events: Com.GetTabset('id').setEvents({ 'tabId' : { 'onShow' : function(tabs, tab){ alert('Example of onShow event'); }, 'onHide' : function(tabs, tab){ alert('Example of onHide event'); } } }); 3.3 Render tabset Com.GetTabset('id').render();
