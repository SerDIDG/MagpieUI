Tabset

  • Initialize in Java-Script:

    new Com.com-tabset(o);
    Attributes:
    • container [HTML Node] - container for tabset, when tabset rendered through java-script, default: cm.Node('div');
    • tabset [HTML Node] - node, that contains tabset, default: false;
    • toggleOnHashChange [Boolean] - use hash change event for toggle tabs, default: true;
    • renderOnInit [Boolean] - render and toggle tabs on his init, default: true;
    • active [String] - id of active tab, default: false;
    • isAdaptive [Boolean] - turn on \ off responsive css, default: true;
    • tabsPosition [String] - set tab position, available: 'top' or 'bottom', default: 'top';
    • showTabs [Boolean] - turn on \ off tabs header, default: true;
    • tabs [Array] - listing of tabs, for rendering through java-script;

    Initialize in HTML:

    <ul class="com-tabset-content-ul" data-tabset="true"> <li data-tabset-id="tab-id" data-tabset-title="Tab Title"></li> </ul> cm.onload(function(){ Com.TabsetCollector(); });
    HTML Attributes, can be set on container node:
    • data-tabset="true" - mark as tabset;
    • data-active="id" - id of active tab;
    • data-toggleOnHashChange="true|false" - use hash change event for toggle tabs;
    • data-renderOnInit="true|false" - render and toggle tabs on his init;
    • data-isAdaptive="true|false" - turn on \ off responsive css;
    • data-tabsPosition="top|bottom" - set tab position;
    • data-showTabs="true|false" - turn on \ off tabs header;
    • data-tabset-id="id" - tab item id;
    • data-tabset-title="title" - tab item title;

    Render tabs through Java-Script:

    new Com.com-tabset({'tabs' : [o]}); or tabset.addTab(o); or Com.GetTabset(id).addTab(o);
    Attributes:
    • id [String] - tab id, default: empty;
    • title [String] - tab title, default: empty;
    • content [HTML Node] - tab content, must be a li tag, default: cm.Node('li');
    • onShow [Function] - event triggered on tab show, default: empty;
    • onHide [Function] - event triggered on tab hide, default: empty;

  • Get tabset by id:

    Com.GetTabset('id');

    Returns a class of tabset.

    Render tabset:

    Render a tabset when he was init with disabled "renderOnInit" attribute. tabset.render();

    Returns a class of tabset.

    Set active tab by id:

    tabset.set(id);

    Returns a class of tabset.

    Get tab by id:

    tabset.get(id);

    Returns tab object.

    Render tab:

    tabset.addTab(o);
    Attributes:
    • id [String] - tab id, default: empty;
    • title [String] - tab title, default: empty;
    • content [HTML Node] - tab content, must be a li tag, default: cm.Node('li');
    • onShow [Function] - event triggered on tab show, default: empty;
    • onHide [Function] - event triggered on tab hide, default: empty;

    Returns a class of tabset.

    Remove tab by id:

    tabset.removeTab(id);

    Returns a class of tabset.

    Set events on tabs:

    Com.GetTabset('id').setEvents({ 'tabId' : { 'onShow' : function(tabset, tab){ alert('Example of onShow event'); }, 'onHide' : function(tabset, tab){ alert('Example of onHide event'); } } });

    Returns a class of tabset.

    Remove tabset:

    tabset.remove();

    Returns a class of tabset.

    Get list of tabset nodes:

    tabset.getNodes(key);

    Returns object of nodes, or node if key is specified.

  • Tabset with onHashChange event:

    Tabset without onHashChange event:

    You can set this attribute in different ways:
    in javascript: new Com.com-tabset({'toggleOnHashChange' : false}); or just add "data-toggleOnHashChange" attribute in html: <ul class="com-tabset-content-ul" data-tabset="true" data-toggleOnHashChange="false"></ul>

    Tabset without onHashChange event and with active tab:

    You can set this attribute by just adding "data-active" attribute in html: <ul class="com-tabset-content-ul" data-tabset="true" data-toggleOnHashChange="false" data-active="second"></ul>

    Tabset with events:

    1. Disable tabset rendering by adding "data-renderOnInit" attribute in html
    <ul class="com-tabset-content-ul" data-tabset="true" data-toggleOnHashChange="false" data-renderOnInit="false"></ul> 2. Set id attribute on tabset in html
    3. Now you can add events in javascript
    3.1 Use Com.GetTabset('id'); function for getting tabset by id.
    Com.GetTabset('id'); 3.2 Add Events: Com.GetTabset('id').setEvents({ 'tabId' : { 'onShow' : function(tabset, tab){ alert('Example of onShow event'); }, 'onHide' : function(tabset, tab){ alert('Example of onHide event'); } } }); 3.3 Render tabset Com.GetTabset('id').render();
