Array

cm.forEach(obj, callback)

The function executes the provided callback once for each element of the array with an assigned value. Returns provided obj.

Parameters

obj
Object to be iterate.
callback
Function to execute for each element.

callback is invoked with two arguments:

  • the element value
  • the element index


cm.isArray(obj)

The function returns true if an obj is array, false if it is not.

Parameters

obj
Object to be checked.


cm.isObject(obj)

The function returns true if an obj is object, false if it is not.

Parameters

obj
Object to be checked.
