Gallery Popup

  • Initialize in Java-Script:

    new Com.GalleryPopup(o);

    Parameters:
    • node [HTML Node] - node, where gallery contains, default: cm.Node('div');
    • duration [Number] - duration of popup opening, default: 300;
    • showCaption [Boolean] - show images caption, default: true;
    • icons [Object] - object of icons css classes;
      • close [String] - css class of close button, default: 'icon medium close-white linked';
    • langs [Object] - object of lang strings;
      • close [String] - popup close title, default: 'Close';
    • Com.Gallery [Object] - Com.Gallery class configuration;

    Simple item html structure:

    <a data-node="items:{}:link" href="images/gallery/1.jpg" title="Sample 1">
    <img src="images/gallery/1-th.jpg" alt="">
</a>

    Parameters:
    • data-node="items:{}:link" - mark item to process by gallery;
    • href - src of full size image / video;
    • title - caption displayed on popup title;

    Initialize in HTML:

    <div class="cm-grid cols-5" data-element="gallery-popup" data-config='{}'>
    <ul>
        <li>
            <div class="cm-imagebox">
                <a data-node="items:{}:link" class="inner" href="images/gallery/1.jpg" title="Sample 1">
                    <img class="descr" src="images/gallery/1-th.jpg" alt="" >
                    <div class="title">
                        <div class="inner">Sample 1</div>
                    </div>
                </a>
            </div>
        </li>
    </ul>
</div>
    var components = new Com.Collector();
components.add('gallery-popup', function(node){
    new Com.GalleryPopup({'node' : node});
});
components.construct();

    Parameters:
    • data-element="gallery-popup" - mark node of gallery container;
    • data-config='{}' - gallery configuration, must be in JSON format.;
    • class="cm-grid cols-x" - where x = from 1 to 12, number of columns;

  • Open popup:

    gallery.open();

    Returns a class of gallery popup.

    Close popup:

    gallery.close();

    Returns a class of gallery popup.

    Set gallery popup active image:

    gallery.set(i);

    Returns a class of gallery popup.

    Set next active gallery popup image:

    gallery.next();

    Returns a class of gallery popup.

    Set previous active gallery popup image:

    gallery.prev();

    Returns a class of gallery popup.

    Add event:

    gallery.addEvent(event, handler);

    Available events: onOpen, onClose, onChange
    Returns a class of gallery popup.

    Remove event:

    gallery.removeEvent(event, handler);

    Available events: onOpen, onClose, onChange
    Returns a class of gallery popup.

  • Gallery listing with rendered html structure:

    1. Add data-element="gallery-popup" attribute on node where gallery contains.
    2. Add data-node="items:[]:link" attribute on each thumbnail link.
    3. To specify popup item title, add title attribute to thumbnail link.
    4. To specify the count of columns, set class name from "cols-1" to "cols-12" on cm-grid node.
    5. To specify gallery configuration, add data-config='{}' attribute on node where you set data-element="gallery-popup" attribute. Configuration must be in JSON format.
    6. Add Com.GalleryPopup class to Collector: 

    var components = new Com.Collector();
components.add('gallery-popup', function(node){
    new Com.GalleryPopup({'node' : node});
});
components.construct();
