Components: Gallery Popup

  • Overview

    This component process server side rendered thumbnails and open popup on each thumbnail click. Supported image files: .jpg, .png, .gif, .jpeg, .bmp, .tga; and video from third-party sites that will be embed in iframe.

    Parameters

    node
    HTML Node. Html node that contains gallery. Default: cm.Node('div').
    size
    String. Size of popup. Available: fullscreen, auto. Default: fullscreen.
    aspectRatio
    String. Aspect ratio of image in gallery. Available: auto, 1x1, 4x3, 3x2, 16x10, 16x9, 2x1, 21x9, 35x10, 3x4, 2x3, 10x16, 9x16, 1x2. Auto - dependence of dimensions when popup size is fullscreen. Default: auto.
    theme
    String. Theme of popup. Available: theme-black, theme-default. Default: theme-black.
    showTitle
    Boolean. Show item title in popup. Default: true.

  • Open popup:

    gallery.open();

    Returns class of gallery popup component.

    Close popup:

    gallery.close();

    Returns class of gallery popup component.

    Set gallery popup active image:

    gallery.set(i);

    Returns class of gallery popup component.

    Set next active gallery popup image:

    gallery.next();

    Returns class of gallery popup component.

    Set previous active gallery popup image:

    gallery.prev();

    Returns class of gallery popup component.

    Set event:

    gallery.addEvent(event, handler);

    Available events: onOpen, onClose, onChange.
    Returns class of gallery popup component.

    Set events:

    gallery.addEvents(o);

    Example: {'onChange' : function(){}}. Available events: onOpen, onClose, onChange.
    Returns class of gallery popup component.

    Remove event:

    gallery.removeEvent(event, handler);

    Available events: onOpen, onClose, onChange.
    Returns class of gallery popup component.

    Trigger event:

    gallery.triggerEvent(event);

    Available events: onOpen, onClose, onChange.
    Returns class of gallery popup component.

  • Gallery listing

    1. To specify popup item title, add title attribute to thumbnail link.
    2. To specify the count of columns, set class name from "cols-1" to "cols-12" on pt__grid node.
    3. To customize popup settings, add data-config='{}' attribute on node where you set data-element="gallery-popup" attribute. Configuration must be in JSON format.
