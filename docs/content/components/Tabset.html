    Parameters:

    container
    HTML Node. Component container that is required in case content is rendered without available tabset node. Default: false.
    node
    HTML Node. Html node that contains tabs. Default: cm.Node('div').
    toggleOnHashChange
    Boolean. Add address bar listener to change tab on hash change. Default: true.
    renderOnInit
    Boolean. After rendering view of tabs, set active tab. Default: true.
    active
    String. Id of tab to be set after rendering view. Default: null.
    className
    String. Add css class to tabset container. Default: empty.
    showTabs
    Boolean. Show tabset header. Default: true.
    tabsPosition
    String. Tabset header position. Available: top, bottom. Default: top.
    tabs

    Array. Listing of tabs, for rendering through java-script. Example:

    events

    Object. Object of events. Available events: onRender, onTabShow, onTabHide. Example:

    icons

    Object. Object of icons CSS class names.

    menu
    String. Menu icon CSS class name. Default: icon default linked.

  • Get tabset by id:

    Com.GetTabset(id);

    Returns class of tabset component.

    Render tabset:

    This method will be init tabset misc helpers and set active tab. Must be called only when parameter renderOnInit is false.

    tabset.render();

    Returns a class of tabset.

    Set active tab by id:

    tabset.set(id);

    Returns class of tabset component.

    Get tab by id:

    tabset.get(id);

    Returns specified tab object.

    Set event:

    tabset.addEvent(event, handler);

    Available events: onRender, onTabShow, onTabHide. Returns class of tabset component.

    Set events:

    tabset.addEvents(o);

    Example: {'onTabShow' : function(tabset, tab){}}. Available events: onRender, onTabShow, onTabHide. Returns class of tabset component.

    Remove event:

    tabset.removeEvent(event, handler);

    Available events: onRender, onTabShow, onTabHide. Returns class of tabset component.

    Trigger event:

    tabset.triggerEvent(event);

    Available events: onRender, onTabShow, onTabHide. Returns class of tabset component.

    Remove tabset:

    This method will be stop all misc helpers and remove tabset html node.

    tabset.remove();

    Returns class of tabset component.

    tabset.getNodes(key);

    Returns object of nodes, or node if key is specified.
