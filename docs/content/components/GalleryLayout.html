Components: Gallery Layout

  • Overview

    This component process server side rendered thumbnails ans embed gallery preview component. Supported image files: .jpg, .png, .gif, .jpeg, .bmp, .tga; and video from third-party sites that will be embed in iframe.

    Required Components

    Initialize


    Parameters

    node
    HTML Node. Html node that contains gallery. Default: cm.Node('div').
    barDirection
    String. Scroll direction of thumbnails bar. Available: horizontal, vertical. Default: horizontal.
    hasBar
    Boolean. If gallery has thumbnails bar, this parameter must be enabled. Default: true.
    events

    Object. Object of events. Available events: onRender, onChange. Example:

  • Set event:

    gallery.addEvent(event, handler);

    Available events: onRender, onChange.
    Returns class of gallery layout component.

    Set events:

    gallery.addEvents(o);

    Example: {'onChange' : function(){}}. Available events: onRender, onChange.
    Returns class of gallery layout component.

    Remove event:

    gallery.removeEvent(event, handler);

    Available events: onRender, onChange.
    Returns class of gallery layout component.

    Trigger event:

    gallery.triggerEvent(event);

    Available events: onRender, onChange.
    Returns class of gallery layout component.
