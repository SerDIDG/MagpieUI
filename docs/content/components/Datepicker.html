Components: Datepicker

  • Overview

    This component decorate input node and convert it to datepicker / datetimepicker. You can specify date format different than input date format.

    Embedded Modules

    • Params
    • Events

    • Langs

    Required Components

    • Com.Select
    • Com.Calendar
    • Com.TimeSelect

    Initialize

    new Com.Datepicker(o);

    or

    Parameters

    container
    HTML Node. Component container that is required in case content is rendered without available input node. Default: false.
    input
    HTML Node. HTML input node. Default: cm.Node('input', {'type' : 'text'}).
    renderInBody
    Boolean. Render datepicker's dropdown in document.body, else it will be rendrered in component container. Default: true.
    format
    String. Input date format. Default: '%Y-%m-%d'.
    displayFormat
    String. Display date format. Default: '%F %j, %Y'.
    isDateTime
    Boolean. Display time selector in datepicker and use date time format. Default: false.
    dateTimeFormat
    String. Input date time format. Default: '%Y-%m-%d %H:%i:%s'.
    displayDateTimeFormat
    String. Display date time format. Default: '%F %j, %Y, %H:%i'.
    minutesInterval
    Number. Minutes interval in time selector. Default: 1.
    startYear
    Number | String. Start year. Default: 1950.
    endYear
    Number | String. End year. Default: 'current + 10'.
    startWeekDay
    Number. First day of week. 0 equal to Sunday, etc. Default: 0.
    showTodayButton
    Boolean. Show button that sets the current date / time. Default: true.
    showClearButton
    Boolean. Show icon outside the input, that clear date / time. Default: false.
    showTitleTooltip
    Boolean. Show title tooltip on hover. Default: true.
    showPlaceholder
    Boolean. Show placeholder text, when date not specified. Default: true.
    title
    String. Title text. Will be shown on hover. Default: empty.
    placeholder
    String. Placeholder text. Default: empty.
    value
    String / Date object. Date / time value in input format or date object or 'now'. Default: 0.
    disabled
    Boolean. Is datepicker disabled. Default: false.
    events

    Object. Object of events. Available events: onRender, onSelect, onChange, onClear, onFocus, onBlur. Example:

    icons

    Object. Object of icons css class names.

    datepicker
    String. Icon css class name, that render inside input area. Icon will be open datepicker on click. Default: icon default linked.
    clear
    String. Icon css class name, that render outside input area. Icon will be clear date / time. Default: icon default linked.
    langs
    Object. Object of language strings.

  • Get datepicker by id:

    Com.GetDatepicker('id');

    Returns class of datepicker component..

    Get value:

    datepicker.get(format);

    Returns value in input format, or in specified format.

    Get value as date object:

    datepicker.getDate();

    Returns value in date object.

    Get selected year:

    datepicker.getFullYear();

    Returns number.

    Get selected month:

    datepicker.getMonth();

    Returns number.

    Get selected date:

    datepicker.getDay();

    Returns number.

    Get selected hours:

    datepicker.getHours();

    Returns number.

    Get selected minutes:

    datepicker.getMinutes();

    Returns number.

    Set value:

    datepicker.set(date);

    Set value in input format, or set value in date object.
    Returns class of datepicker component.

    Set event:

    datepicker.addEvent(event, handler);

    Available events: onRender, onSelect, onChange, onClear, onFocus, onBlur.
    Returns class of datepicker component.

    Set events:

    datepicker.addEvents(o);

    Example: {'onChange' : function(){}}. Available events: onRender, onSelect, onChange, onClear, onFocus, onBlur. Returns class of datepicker component.

    Remove event:

    datepicker.removeEvent(event, handler);

    Available events: onRender, onSelect, onChange, onClear, onFocus, onBlur.
    Returns class of datepicker component.

    Trigger event:

    datepicker.triggerEvent(event);

    Available events: onRender, onSelect, onChange, onClear, onFocus, onBlur. Returns class of datepicker component.

    Disable:

    datepicker.disable();

    Returns class of datepicker component.

    Enable:

    datepicker.enable();

    Returns class of datepicker component.

    Get list of datepicker nodes:

    datepicker.getNodes(key);

    Returns object of nodes, or node if key is specified.
