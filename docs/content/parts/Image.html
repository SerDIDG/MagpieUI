  • Resizable Image Box

    Image is resized by available parent block width preserving own aspect ratio.

    Adaptive Image Box

    Image is resized by available parent block width but not more than its own width preserving own aspect ratio.

    Flexible Image Box

    Image is resized by available parent block width and height. For example was set parent block aspect ratio 4x3.

    Available Aspect Ratios. Landscape: 4x3, 3x2, 16x10, 16x9, 21x9, 35x10. Portrait: 3x4, 2x3, 10x16, 9x16.

    Centered Image Box

    Image is resized by available parent block width and height but not more than its own width and height preserving own aspect ratio and align center of parent block. For example was set parent block aspect ration 4x3.

    Available Aspect Ratios. Landscape: 4x3, 3x2, 16x10, 16x9, 21x9, 35x10. Portrait: 3x4, 2x3, 10x16, 9x16.
