Glossary

  • Initialize in Java-Script:

    new Com.Glossary(o);

    Attributes:
    • glossary [HTML Node] - node of glossary item (.com-glossary-word), default: false;

    Initialize in HTML:

    <span class="com-glossary-word" data-com-glossary="true">
    <span class="com-glossary-title">Glossary Word</span>
    <span class="com-glossary-popup">Glossary Description</span>
</span>
    cm.onload(function(){
    Com.GlossaryCollector(/* container, attribute, value */);
});

    HTML Attributes:
    • data-com-glossary="true" - mark as glossary, can be customize in initialization of glossary collector;

  • Get glossary by id:

    Com.GetGlossary('id');

    Returns a class of glossary.

    Unlink and destroy glossary class:

    glossary.remove();

    Returns a class of glossary.
