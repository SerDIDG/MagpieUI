Datepicker

  • Initialize in Java-Script:

    new Com.Datepicker(o);

    Attributes:
    • container [HTML Node] - container for select, when select rendered through java-script, default: cm.Node('div');
    • input [HTML Node] - input type text html node, default: cm.Node('input', {'type' : 'text'});
    • placeholder [String] - placeholder text, default: empty;
    • format [String] - date format for displays in input, default: '%F %j, %Y';
    • saveFormat [String] - date format for saving on the server, default: '%Y-%m-%d';
    • startYear [Number] - minimal year that is displays in datepicker, default: 1900;
    • endYear [Number] - maximal year that is displays in datepicker, default: current;
    • startWeekDay [Number] - specify the day which the week starts, default: 0;
    • showPlaceholder [Boolean] - show placeholder text, when date not specified, default: true;
    • showTodayButton [Boolean] - display a button that sets the current date, default: true;
    • showClearButton [Boolean] - display a icon outside the input, that is clear input, default: false;
    • showTitleTag [Boolean] - show title on hover, default: true;
    • title [String] - title text, default: false;
    • menuMargin [Number] - outer margin of datepicker window, default: 3;
    • langs [Object] - object of lang strings;
      • days [Array] - abbreviation of days, default: ['S', 'M', 'T', 'W', 'T', 'F', 'S'];
      • months [Array] - name of months, default: ['January', 'February', 'March', 'April', 'May', 'June', 'July', 'August', 'September', 'October', 'November', 'December'];
      • clearButtonTitle [String] - title tag, that is displays on hover clear button, default: 'Clear datepicker';
      • todayButton [String] - text, that is displays on today button, default: 'Today';

    Attributes:
    • data-datepicker="true|false" - mark as datepicker, if false - datepicker will not render;
    • data-title or title - title text;
    • data-showTitleTag="true|false" - show title on hover;
    • data-placeholder or placeholder - placeholder text;
    • data-showPlaceholder="true|false" - show placeholder text, when date not specified;
    • data-showTodayButton="true|false" - display a button that sets the current date;
    • data-showClearButton="true|false" - display a icon outside the input, that is clear input;
    • data-startYear - minimal year that is displays in datepicker;
    • data-endYear - maximal year that is displays in datepicker;
    • data-startWeekDay - specify the day which the week starts;
    • data-format - date format for displays in input;
    • data-saveFormat - date format for saving on the server;

  • Get datepicker by id:

    Com.GetDatepicker('id');

    Returns a class of datepicker.

    Get selected date string in save format:

    datepicker.get(format);

    Returns selected date string in save (server) format, or in specified format.

    Set date:

    datepicker.set(date);

    Set date by specify date string in save (server) format or date object.
    Returns a class of datepicker.

    Get selected date object:

    datepicker.getDate();

    Returns selected date object.

    Set event:

    datepicker.addEvent(event, handler);

    Available events: onSelect, onChange.
    Returns a class of datepicker.

    Remove event:

    datepicker.removeEvent(event, handler);

    Available events: onSelect, onChange.
    Returns a class of datepicker.

    Convert date object to date string:

    datepicker.parseDate(o, format);

    Returns date string in save (server) format, or in specified format.

    Get list of datepicker nodes:

    datepicker.getNodes(key);

    Returns object of nodes, or node if key is specified.
