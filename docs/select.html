Select

  • Initialize in Java-Script:

    new Com.Select(o);
    Attributes:
    • container [HTML Node] - container for select, when select rendered through java-script, default: cm.Node('div');
    • select [HTML Node] - select html node, default: cm.Node('select');
    • multiple [Boolean] - use multiple select, default: false;
    • showTitleTag [Boolean] - show title on hover, default: true;
    • title [String] - title text, default: false;
    • menuMargin [Number] - outer margin of select menu, default: 3;
    • options [Array] - listing of options, for rendering through java-script;
    • selected [String/Array] - id / array of selected item / items, default: 0;

    Initialize in HTML:

    <select></select> cm.onload(function(){ Com.SelectCollector(); });
    Attributes:
    • data-select="true|false" - mark as select, if false - select will not render;
    • data-multiple or multiple - use multiple select;
    • data-title or title - title text;
    • data-showTitleTag="true|false" - show title on hover;

    Render options through Java-Script:

    new Com.Select({'options' : [{'value' : '', 'text' : ''}]}); or select.addOption(value, text) or select.addOptions([{'value' : '', 'text' : ''}]); or Com.GetSelect(id).addOption(value, text); or Com.GetSelects(id).addOptions([{'value' : '', 'text' : ''}]);

  • Get select by id:

    Com.GetSelect('id');

    Returns a class of select.

    Get selected option value:

    select.get();

    Returns selected option value, or array of values if select is multiple.

    Select option value (set active option):

    select.set(value);

    Set active option value, or array of values if select is multiple.
    Returns a class of select.

    Set event:

    select.addEvent(event, handler);

    Available events: onSelect, onChange, onFocus, onBlur.
    Returns a class of select.

    Remove event:

    select.removeEvent(event, handler);

    Available events: onSelect, onChange, onFocus, onBlur.
    Returns a class of select.

    Add option:

    select.addOption(value, text);

    Returns a class of select.

    Remove option:

    select.removeOption(value);

    Returns a class of select.

    Add options:

    select.addOptions([{'value' : '', 'text' : ''}]);

    Returns a class of select.

    Remove all options:

    select.removeOptionAll();

    Returns a class of select.

    Get list of select nodes:

    select.getNodes(key);

    Returns object of nodes, or node if key is specified.
