Gallery

  • Initialize in Java-Script:

    new Com.Gallery(o);
    Attributes:
    • gallery [HTML Node] - node, that contains links with thumbnails, default: false;
    • minWidth [Number] - default overlay width in px, default: 400;
    • minHeight [Number] - default overlay height in px, default: 400;
    • openTime [Number] - time of open overlay in ms, default: 300;
    • margin [Number] - outer margin of overlay for desktop browsers in px, default: 70;
    • marginMobile [Number] - outer margin of overlay for mobile browsers in px, default: 10;
    • slideChangeTime [Number] - time of change slide into overlay in ms, default: 300;
    • showArrowTitles [Boolean] - show overlay arrows titles on hover, default: false;
    • showTitle [Boolean] - show images captions into overlay title, default: true;
    • showCounter [Boolean] - show images counter into overlay title, default: true;
    • langs [Object] - object of lang strings;
      • next [String] - overlay right arrow title, default: 'Next';
      • prev [String] - overlay left arrow title, default: 'Previous';
      • close [String] - overlay close title, default: 'Close';
      • counter [String] - overlay images counter, default: '%item% / %items%';

    Image item html structure:

    <a href="images/gallery/1.jpg" data-image="true"> <img src="images/gallery/1-th.jpg" alt="Sample 1" title="Sample 1"> </a>
    Attributes:
    • data-image="true" - mark as gallery item;
    • href - src of full size image;
    • title - image caption for overlay title;

    Initialize in HTML:

    <div class="com-gallery-listing cols-6" data-gallery="true"> <ul> <li> <a href="images/gallery/1.jpg" data-image="true"> <img src="images/gallery/1-th.jpg" alt="Sample 1" title="Sample 1"> </a> </li> </ul> </div> cm.onload(function(){ Com.GalleryCollector(); });
    Attributes:
    • data-gallery="true" - mark as gallery album;
    • data-showArrowTitles="true|false" - show overlay arrows titles on hover;
    • data-showTitle="true|false" - show images captions into overlay title;
    • data-showCounter="true|false" - show images counter into overlay title;
    • class="gallery-listing cols-x" - where x = from 1 to 10, number of columns;

  • Get gallery by id:

    Com.GetGallery('id');

    Return a class of gallery.

    Open gallery overlay:

    gallery.open();

    Return a class of gallery.

    Close gallery overlay:

    gallery.close();

    Return a class of gallery.

    Set gallery overlay active image:

    gallery.set(i);

    Return a class of gallery.

    Set next active gallery overlay image:

    gallery.next();

    Return a class of gallery.

    Set previous active gallery overlay image:

    gallery.prev();

    Return a class of gallery.

    Add event:

    gallery.addEvent(event, handler);

    Available events: onOpen, onClose, onImageLoad
    Return a class of gallery.

    Remove event:

    gallery.removeEvent(event, handler);

    Available events: onOpen, onClose, onImageLoad
    Return a class of gallery.

  • Gallery listing with rendered html structure:

    1. Add "data-gallery" attribute on node, and "data-image" on links with thumbnails.
    1.1 For adding overlay title, add title attribute on thumbnail.
    1.2 To specify the count of columns, set class name from "cols-1" to "cols-10".
    2. Init gallery collector: Com.GalleryCollector();
