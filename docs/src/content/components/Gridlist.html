Components: Gridlist

  • Overview

    Component for rendering tables of data - gridlists. Supported dynamic pagination, bulk actions, resizable columns, etc.

    Embedded Modules

    • Params
    • Events
    • Langs
    • Stack

    Required Components

    • Com.GridlistHelper - for resizable columns.
    • Com.Pagination - for pagination.
    • Com.Toolbar - for bulk actions.
    • Com.Menu - for dropdown actions.

    Initialize

    Java-script

    Auto Collector

    Required Com.Collector with parameter "autoInit" : true.

    Parameters

    Default

    node
    HTML Node. Html node for embedding gridlist. Default: cm.node('div').
    name
    String. Name of class instance. Default: none.

    Data

    uniqueKey
    String. Unique data key, uses for indexing. Default: id.
    data

    Array. Array of data. Default: [].

    _checked
    Boolean. Mark row as checked. Default: none.
    _status
    String. Mark row with status. Available: active, success, danger, warning. Default: none.
    cols

    Array. Array of columns. Default: [].

    key
    String. Key of data array, all cells of this column will be display content by this key. Default: none.
    title
    String. Title of column. Default: none.
    width
    String/Number. Width of column, can be number or percentage or 'auto'. Default: auto.
    access
    Boolean. Toggle column rendering, if false, column will not be rendered. Default: true.
    type

    String. Type of column content, needs to be specified for properly sorting and displaying. Default: text.

    • text - for text rendering;
    • number - for number rendering;
    • url - for links rendering;
    • date - for date rendering;
    • html - for rendering html and properly sort html text nodes;
    • icon - for rendering icon node, sorting not available;
    • checkbox - for rendering checkbox node, sorting not available;
    • empty - for rendering empty content cells, sorting not available;
    • actions - for rendering actions dropdown;
    • links - for rendering actions menu.
    sort
    Boolean. Disable/Enable sorting in this column. Default: as global.
    altText
    String. Displays alternative text in cell instead of data. Default: none.
    urlKey
    String. Insert href attribute by this key, when column type is url. Default: none.
    showTitle
    Boolean. Show title on hover, by default using content of data key. Default: false.
    titleText
    String. Alternative title text on hover. Default: none.
    textOverflow
    Boolean. Disable/Enable text no-wrap. Default: false.
    preventDefault
    Boolean. Prevent default click/hover event. Default: true.
    target
    String. Open links target, when column type is url type is icon. Default: _blank.
    class
    String. CSS class for icon node, when column type is icon. Default: none.
    onClick
    Function. On cell click event handler. When type is icon url checkbox, event will be bind on this nodes. Default: none.
    onRender
    Function. On cell render event handler. Default: none.
    actions

    Array. Array of actions for action dropdown, when column type is actions. Default: [].

    label
    String. Label of link.
    attr
    Object. Associative array of html attributes, value can contains variables, example %id%
    constructor
    String. Class Constructor.
    constructorParams
    Object. Class parameters
    callback
    Function. Onclick callback.
    links

    Array. Array of actions for links menu, when column type is links. Default: [].

    label
    String. Label of link.
    attr
    Object. Associative array of html attributes, value can contains variables, example %id%
    events
    Object. Associative array of dom events.
    actions

    Array. Array of bulk actions. Default: [].

    label
    String. Label of link.
    title
    String. Tile on hover.
    permanent
    Boolean. Always enable action. Default: false.
    type
    String. Style of button. Available: primary, secondary, success, danger, warning. Default: primary:
    className
    String. CSS class. Default: none.
    constructor
    String. Class Constructor.
    constructorParams
    Object. Class parameters
    callback
    Function. Onclick callback.

    Sorting

    sort
    Boolean. Enable/Disable sorting. Default: true.
    sortBy
    String. Default sorting column by key. Default: id.
    orderBy
    String. Default: ASC.
    childsBy
    Boolean. Render child rows after parent, (WIP - doesn't work checking / uncheking rows and statuses for now). Default: false.

    Visibility

    showCounter
    Boolean. Show items counter. Default: false.
    showBulkActions
    Boolean. Show toolbar with bulk actions. Default: true.
    className
    String. CSS class. Default: none.
    dateFormat
    String. Input date format. Default: 'cm._config.dateTimeFormat'.
    visibleDateFormat
    String. Display date format. Default: 'cm._config.dateTimeFormat'.

    Pagination and ajax data request

    pagination
    Boolean. Enable/Disable pagination. Default: true.
    perPage
    Number. Rows per page. Default: 25.
    responseKey
    String. Response data response key. Default: data.
    responseCountKey
    String. Response data count response key. Default: count.
    ajax
    Object. Ajax request object. Default: {}.

    Columns manipulation

    columns

    Object.

    editable
    Boolean. Enable/Disable resizable columns. Default: false.
    ajax
    Object. Ajax request object. Default: {}.

  • Add event:

    gridlist.addEvent(event, handler);

    Available events: onSort, onCheckAll, onUnCheckAll, onCheck, onUnCheck, onRenderStart, onRenderEnd.
    Returns a class of gridlist.

    Remove event:

    gridlist.removeEvent(event, handler);

    Available events: onSort, onCheckAll, onUnCheckAll, onCheck, onUnCheck, onRenderStart, onRenderEnd.
    Returns a class of gridlist.

    Check row:

    gridlist.check(id);

    index [Nubmer] - id of row in data.
    Trigger onCheck event, which returns: [Object {Gridlist}, Object {row}].
    Returns a class of gridlist.

    Uncheck row:

    gridlist.unCheck(id);

    index [Nubmer] - id of row in data.
    Trigger onUnCheck event, which returns: [Object {Gridlist}, Object {row}].
    Returns a class of gridlist.

    Check all rows:

    gridlist.checkAll();

    Trigger onCheckAll event, which returns: [Object {Gridlist}, Object {data}].
    Returns a class of gridlist.

    Uncheck all rows:

    gridlist.unCheckAll();

    Trigger onUnCheckAll event, which returns: [Object {Gridlist}, Object {data}].
    Returns a class of gridlist.

    Get checked rows:

    gridlist.getChecked();

    Returns the object of rows.

    Get checked rows data:

    gridlist.getCheckedData();

    Returns the object of rows data.

    Get checked rows indexes:

    gridlist.getCheckedIndexes();

    Returns the array of indexes.

    Set row status:

    gridlist.setRowStatus(id, status);

    index [Nubmer] - id of row in data.
    status [String] - status name. Available: active, success, danger, warning.
    Returns a class of gridlist.

    Clear row status:

    gridlist.clearRowStatus(id);

    index [Nubmer] - id of row in data.
    Returns a class of gridlist.

    Get rows by status:

    gridlist.getRowsByStatus(status);

    status [String] - status name. Available: active, success, danger, warning.
    Returns the object of rows.
